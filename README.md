Este projeto é desenvolvido como parte prática da disciplina de Engenharia de Software na Universidade Cruzeiro do Sul. O objetivo é aplicar conceitos avançados de modelagem, rastreabilidade e arquitetura de sistemas no desenvolvimento de uma aplicação real de economia compartilhada.

## O que é o Aluga Fácil?

Seguindo as diretrizes da instituição, este projeto foi idealizado para solucionar uma necessidade real de mercado. O **Aluga Fácil** foca na otimização de recursos através da economia compartilhada.

> **Ideia Central:** Facilitar a conexão entre pessoas que possuem itens subutilizados e pessoas que precisam desses produtos temporariamente, promovendo um consumo mais inteligente, sustentável e acessível.

### Principais Dores Resolvidas:
* **Custo de Aquisição:** Elimina a necessidade de compra de itens caros para usos esporádicos.
* **Acúmulo de Bens:** Reduz o desperdício e a ocupação de espaço com objetos sem uso frequente.
* **Renda Extra:** Permite que proprietários monetizem seus ativos parados.
